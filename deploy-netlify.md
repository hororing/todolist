# Netlify 배포 가이드

## 1단계: Netlify 계정 생성
1. [Netlify.com](https://netlify.com)에 가입하거나 로그인

## 2단계: 사이트 배포
### 방법 A: 드래그 앤 드롭 (가장 간단)
1. Netlify 대시보드에서 "Sites" 섹션으로 이동
2. `index.html`이 있는 폴더 전체를 드래그해서 Netlify에 드롭
3. 자동으로 배포 완료!

### 방법 B: GitHub 연동
1. "New site from Git" 클릭
2. GitHub 선택 후 저장소 연결
3. 자동 배포 설정

## 3단계: 도메인 설정
- 기본 도메인: `https://random-name.netlify.app`
- 커스텀 도메인 설정 가능 (무료)

## 장점
- 매우 간단한 배포
- 자동 HTTPS
- 글로벌 CDN
- 폼 처리 기능
